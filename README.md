# community_toolkit

Community Toolkit for Flutter applications.

This package currently includes the extracted MVVM framework that lived in
`lib/mvvm/` in the app:

- `BaseViewModel`
- `RelayCommand`
- `Bind` / `BindSelector` / `BindCommand`
- `Messenger`

## Install

```yaml
dependencies:
  community_toolkit:
    path: ../community_toolkit
```

## Import

```dart
import 'package:community_toolkit/community_toolkit.dart';
```

## Feature Entry Points

- `package:community_toolkit/community_toolkit.dart`
- `package:community_toolkit/mvvm.dart`

## Example

Run the example app from `example/`.
